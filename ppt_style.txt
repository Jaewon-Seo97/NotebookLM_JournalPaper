[발표자료 공통 스타일 규격]  ── 콘텐츠 지시 뒤에 append. 내용이 바뀌어도 이 룩을 유지한다.

■ 산출물 (항상 두 파일을 함께 생성)
1) slides.md   — 편집용 구조화 원본. 내용의 단일 원천(single source of truth). 아래 MD 스키마를 따른다.
2) slides.pptx — md와 내용이 100% 일치하는 렌더링본. 16:9(13.333"×7.5"), python-pptx로 생성.
   ※ md가 원천, pptx는 그 시각화. 두 파일의 텍스트·수치·인용은 항상 동일해야 한다.

■ MD 스키마 (슬라이드마다 반복)
## Slide N — {대제목}
- Section:  {좌측 사이드바 섹션명}
- Kicker:   {오버라인: "대섹션 - 소주제"}
- Subtitle: {부제 1줄 = 이 페이지의 목적}
- Bulletins (3~4개):
  - [01] {소제목} · {저자, 연도} — {2~3문장 근거}
  - [02] ...
- Panels:
  - [A] {헤더} — {그림 경로 또는 설명} — {캡션 1줄}
  - [B] ...
- Callouts:
  - {라벨} : {값}  (color=gray|orange|navy|red★)
- Definition(opt): {용어} = {한 문장 정의}
- Key Message: {하단 결론바 1문장}
- References: {저자, 저널, 연도} …

■ 컬러 토큰 (지정 값만 사용)
Navy #1F3864(주색·대제목·헤더바·결론바) / Navy-teal #37657B(헤더 교차)
Gold/Tan #C6A06F(DEFINITION 라벨·헤더 교차·구분선) / Orange #ED7D31(관심 데이터 1계열)
Red #C00000(★ 최고 강조, 페이지당 1곳) / Gray #8E97A8(비교군·baseline)
Sidebar #CBE6F2→#EAF3F8 그라데이션 / 본문배경 #FFFFFF
텍스트: 밝은 배경 #1A1A1A, 어두운 바 위 #FFFFFF

■ 타이포그래피
Serif(제목·소제목·본문 강조·번호): "Noto Serif KR", Georgia, "Times New Roman"
Sans(오버라인·차트축·사이드바·캡션): "Noto Sans KR", Arial, Helvetica
번호 01·02·03 = Serif Italic / 대제목 = Serif Bold 특대
(한글 폰트 미탑재 시 pptx는 바탕/맑은 고딕으로 대체 지정)

■ 고정 레이아웃 골격
1) 좌측 사이드바(폭 ~13%): 상단 큰 섹션명 1개 + 하단 목차(현재 항목만 진한 Navy, 나머지 흐린 회청).
2) 헤더: 오버라인 kicker(소형 bold sans Navy) → 대제목(Serif Bold 특대) → 부제(Serif 1줄).
3) 요약 bulletin 3~4개 가로 정렬: 번호 + 소제목·핵심어 + 인용(저자 연도) + 2~3문장.
4) 그림 패널 A·B·C: 상단 컬러 헤더바(레터 태그+제목, Navy/Tan 교차), 내부는 그림·도표 위주 + 1줄 캡션. 화면의 절반 이상 차지.
5) 데이터 콜아웃: 수치 강조 박스, 색상 코딩(회색=비교군, 주황/파랑=관심, 빨강★=최고 결과 1개), 값 옆 의미 라벨.
6) DEFINITION 박스(필요 시): Tan 라벨 + Navy 본문, 핵심 용어 1개 정의.
7) 하단 결론바(필수): Navy 전폭, "→" + 굵은 결론 1문장.

■ 필수 콘텐츠 슬롯 (4요소)
목적→헤더(오버라인·부제) / 핵심 메시지→하단 결론바 1문장 / 근거→bulletin·패널·콜아웃에 (저자, 연도) 병기 / 참고자료→인라인 인용 + 하단 각주.

■ 스타일 규칙
- 그림:텍스트 ≥ 6:4, 도표·이미지가 화면 절반 이상. 텍스트 밀도 낮게, 여백 확보.
- bulletin 3~4개, 각 2~3문장 이내.
- 모든 수치에 (저자, 연도). 출처 없는 수치·추정치·"약 ~%"·"많은 연구에서" 금지. 없으면 "No quantitative estimate reported".
- Red(★)는 페이지당 1곳만. 패널 헤더바는 Navy↔Tan 교차.
- 캡션·라벨은 Sans, 서술문은 Serif로 통일.

■ PPTX 렌더링 규칙
- 슬라이드 = 16:9. 위 1600×900 좌표를 13.333"×7.5" 비율로 매핑.
- 각 요소를 개별 도형/텍스트박스로 생성해 편집 가능하게. 색·폰트는 위 토큰 그대로.
- 그림은 파일 경로로 삽입. 경로 없으면 회색 placeholder 박스 + 캡션 유지.

■ 자체 점검(출력 전)
□ slides.md와 slides.pptx의 내용·수치·인용이 일치하는가?
□ 사이드바(섹션명+목차, 현재 위치 강조)가 있는가?
□ 헤더 3층(오버라인·대제목·부제)이 있는가?
□ bulletin 3~4개가 인용과 함께 있는가?
□ 그림 패널이 화면 절반 이상인가?
□ 하단 결론바에 핵심 메시지 1문장이 있는가?
□ 모든 수치에 (저자, 연도)가 있는가?
□ 지정 컬러 토큰·폰트만 사용했는가?  □ PPTX가 16:9인가?
